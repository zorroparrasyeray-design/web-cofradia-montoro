# 🕊️ Cofradía del Stmo. Cristo Resucitado de Montoro

## Sitio Web Oficial

Web moderna e interactiva de la Cofradía del Santísimo Cristo Resucitado de Montoro.

### 📋 Contenido

- Información general de la cofradía
- Historia y patrimonio
- Junta de gobierno
- Calendario de cultos y actividades
- Estación de Gloria (Domingo de Resurrección)
- Galería multimedia
- Panel de administrador
- Sistema de nuevo hermano
- Tienda de merchandising
- Información de contacto

### 🎨 Características

- Diseño moderno y responsivo
- Colores oficiales: Blanco, Marfil y Rojo
- Panel de administrador para gestión de contenidos
- Integración de mapas interactivos
- Elementos 3D patrimoniales
- Sistema de noticias categorizado
